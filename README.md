## ✨ Overview

This is my personal portfolio website — designed to be sleek, fast, and interactive, with a dark, premium aesthetic. It highlights my projects, skills, and provides a direct way for people to reach out and collaborate.

## 🚀 Features

- **Animated Hero & Sections** — smooth scroll-triggered reveal animations built with Framer Motion
- **Interactive Contact Form** — messages are sent directly to my inbox via [Web3Forms](https://web3forms.com), no backend required
- **Direct Social Links** — one-click access to Gmail, WhatsApp, GitHub, LinkedIn, and LeetCode
- **Premium Dark UI** — glassmorphism cards, glow effects, animated gradients, and a subtle grid background
- **Fully Responsive** — optimized layouts for mobile, tablet, and desktop
- **Fast & Lightweight** — built on Vite for near-instant load times

## 🛠️ Tech Stack

## ⚙️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Krishnna1909/portfolio.git

# Navigate into the project
cd portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

The site will be running at `http://localhost:5173`.

### Build for Production

```bash
npm run build
```

## 🔑 Environment Variables

This project uses [Web3Forms](https://web3forms.com) for the contact form. Create a `.env` file in the root directory (or update directly in `ContactSection.tsx`):

```
VITE_WEB3FORMS_ACCESS_KEY=your_access_key_here
```

Get your free access key at [web3forms.com](https://web3forms.com) — no signup required.
